# Email Template Visual Guidelines

*Crowdfunding Campaign Email Design System*

## Email Design Strategy

**Purpose**: Create consistent, professional email templates that drive engagement and conversions  
**Target Audience**: Tech-savvy professionals and families interested in smart home solutions  
**Brand Voice**: Sophisticated, trustworthy, clear, benefit-focused  
**Design Philosophy**: Clean, mobile-first, conversion-optimized

## Visual Design Principles

### Brand Consistency
- Maintain HeyZack visual identity across all emails
- Use consistent color palette and typography
- Include proper logo placement and sizing
- Ensure brand voice alignment in visual elements

### Mobile-First Design
- Single-column layouts for optimal mobile viewing
- Minimum 44px touch targets for buttons
- Readable font sizes (minimum 16px for body text)
- Optimized images for fast loading

### Conversion Optimization
- Clear visual hierarchy guiding to CTAs
- Strategic use of white space
- Prominent, contrasting call-to-action buttons
- Scannable content with visual breaks

## Typography System

### Primary Fonts

**Headlines**: Brinnan-Bold
- H1: 32px, line-height 1.2
- H2: 28px, line-height 1.3
- H3: 24px, line-height 1.4

**Subheadings**: Brinnan-Light
- Large: 22px, line-height 1.4
- Medium: 20px, line-height 1.4
- Small: 18px, line-height 1.5

**Body Text**: Avenir-Light
- Primary: 16px, line-height 1.6
- Secondary: 14px, line-height 1.5
- Caption: 12px, line-height 1.4

### Font Hierarchy Rules

1. **Email Subject Lines**: Brinnan-Bold, sentence case
2. **Main Headlines**: Brinnan-Bold, title case
3. **Section Headers**: Brinnan-Light, title case
4. **Body Content**: Avenir-Light, sentence case
5. **CTAs**: Brinnan-Bold, ALL CAPS
6. **Fine Print**: Avenir-Light, 12px minimum

**HeyZack Orange**: #FF6B35
- Usage: CTAs, accents, highlights
- Accessibility: AA compliant on white

- Usage: Backgrounds, negative space

- Primary: Clean White (#FFFFFF)

**Text Colors**:
- Primary: HeyZack Charcoal (#1D1D1B)
- Secondary: Cool Gray (#8E8E93)

## Call-to-Action (CTA) Design

### Primary CTA Button

**Specifications**:
```
Background: HeyZack Orange (#FF6B35)
Text: White (#FFFFFF)
Font: Brinnan-Bold, 16px, ALL CAPS
Padding: 16px horizontal, 12px vertical
Border-radius: 6px
Minimum width: 200px
Minimum height: 44px (mobile touch target)
```

**Hover State**:
```
Background: Darker orange (#E55A2B)
Text: White (no change)
Transition: 0.2s ease
```

### Secondary CTA Button

**Specifications**:
```
Background: HeyZack Charcoal (#1D1D1B)
Text: White (#FFFFFF)
Font: Brinnan-Bold, 16px, ALL CAPS
Padding: 16px horizontal, 12px vertical
Border-radius: 6px
```

### Ghost CTA Button

**Specifications**:
```
Background: Transparent
Border: 2px solid HeyZack Orange (#FF6B35)
Text: HeyZack Orange (#FF6B35)
Font: Brinnan-Bold, 16px, ALL CAPS
Padding: 14px horizontal, 10px vertical
Border-radius: 6px
```

### CTA Placement Rules

1. **Above the fold**: Primary CTA within first 600px
2. **Repetition**: Repeat primary CTA every 3-4 sections
3. **Hierarchy**: One primary, multiple secondary CTAs allowed
4. **Spacing**: Minimum 40px margin around CTAs
5. **Mobile**: Full-width on screens under 480px

## Layout Structure

### Email Width Standards

**Desktop**: 600px maximum width  
**Mobile**: 100% width with 20px side margins  
**Content**: 560px maximum (accounting for 20px padding)

### Section Spacing

**Between sections**: 40px  
**Within sections**: 20px  
**Around CTAs**: 40px minimum  
**Header/footer margins**: 30px

### Grid System

**Single Column**: Default for all emails  
**Two Column**: Only for desktop, stacks on mobile  
**Three Column**: Avoid - poor mobile experience

## Image Guidelines

### Image Specifications

**Header Images**: 600px x 300px maximum  
**Product Images**: 400px x 400px (square format)  
**Feature Graphics**: 560px x 280px (2:1 ratio)  
**Icons**: 60px x 60px (consistent sizing)

### Image Optimization

**File Format**: JPG for photos, PNG for graphics with transparency  
**File Size**: Maximum 200KB per image  
**Alt Text**: Required for all images  
**Fallback**: Background color for slow loading

### Image Style Guidelines

**Product Photography**:
- Clean, minimal backgrounds
- Consistent lighting and shadows
- Multiple angles when beneficial
- Lifestyle context when appropriate

**Graphics and Icons**:
- Consistent style (outlined or filled)
- HeyZack color palette
- Scalable for different sizes
- Clear, recognizable symbols

## Accessibility Standards

### Color Contrast

**Text on White**: Minimum 4.5:1 ratio (AA standard)  
**Large Text**: Minimum 3:1 ratio  
**CTAs**: Minimum 4.5:1 ratio for text

### Font Sizes

**Minimum Body Text**: 16px  
**Minimum Secondary Text**: 14px  
**Minimum Fine Print**: 12px

### Interactive Elements

**Touch Targets**: Minimum 44px x 44px  
**Link Identification**: Color + underline or button style  
**Focus States**: Visible outline for keyboard navigation
