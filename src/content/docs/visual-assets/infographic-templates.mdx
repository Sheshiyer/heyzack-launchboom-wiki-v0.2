- **Bounces**: cubic-bezier for playfulness

## Usage Guidelines

### Platform Adaptations

**Social Media**:
- **Instagram**: Square (1080x1080px)
- **Facebook**: Landscape (1200x630px)
- **Twitter**: Landscape (1024x512px)
- **LinkedIn**: Landscape (1200x627px)
- **Stories**: Vertical (1080x1920px)

**Email Marketing**:
- **Header**: 600px width maximum
- **Inline**: 300px width maximum
- **Footer**: 600px width maximum
- **Mobile**: Responsive scaling

**Web Usage**:
- **Hero Sections**: Full-width responsive
- **Content Blocks**: Container-width
- **Sidebars**: 300px fixed width
- **Modals**: Centered, responsive

### Quality Standards

**Resolution Requirements**:
- **Print**: 300 DPI minimum
- **Web**: 72 DPI standard, 144 DPI retina
- **Social**: Platform-specific optimization
- **Email**: 72 DPI, optimized file size

**File Formats**:
- **Web**: SVG preferred, PNG fallback
- **Print**: PDF or high-res PNG
- **Social**: JPG for photos, PNG for graphics
- **Email**: JPG optimized for size

### Brand Compliance

**Logo Usage**:
- **Minimum Size**: 100px width
- **Clear Space**: 2x logo height
- **Backgrounds**: Ensure sufficient contrast
- **Modifications**: No alterations allowed

**Voice & Tone**:
- **Professional**: Technical accuracy
- **Approachable**: Friendly explanations
- **Confident**: Clear value propositions
- **Trustworthy**: Honest representations

These infographic templates provide a comprehensive visual communication system that effectively conveys HeyZack's value propositions while maintaining brand consistency and professional presentation across all marketing channels and customer touchpoints.
