## Design System Guidelines

### Color Palette

**Primary Colors**:
- **HeyZack Blue**: #1B2951 (Trust, technology)
- **Smart Green**: #2A9D8F (Efficiency, growth)
- **Energy Orange**: #F4A261 (Savings, warmth)
- **Clean White**: #FFFFFF (Clarity, space)
- **Neutral Gray**: #6C757D (Balance, sophistication)

**Accent Colors**:
- **Success Green**: #28A745
- **Warning Orange**: #FFC107
- **Error Red**: #DC3545
- **Info Blue**: #17A2B8

### Typography System

**Primary Font**: Montserrat
- **Headers**: Montserrat Bold (700)
- **Subheaders**: Montserrat SemiBold (600)
- **Body**: Montserrat Regular (400)
- **Captions**: Montserrat Light (300)

**Secondary Font**: Open Sans
- **Body Text**: Open Sans Regular (400)
- **Small Text**: Open Sans Light (300)
- **Technical**: Open Sans Mono (400)

### Icon Library

**Core Icons**:
- 🏠 Home/House
- 🧠 AI/Brain
- 🛡️ Security/Shield
- 💡 Energy/Lightbulb
- 🗣️ Voice/Speech
- 📱 Mobile/App
- ⚙️ Settings/Gear
- 📊 Analytics/Chart
- 🔒 Privacy/Lock
- ✅ Success/Check

**Style Guidelines**:
- **Line Weight**: 2px consistent
- **Corner Radius**: 4px for rounded elements
- **Size**: 24px, 32px, 48px standard sizes
- **Color**: Match brand palette
- **Style**: Outline preferred, filled for emphasis

### Layout Principles

**Grid System**:
- **Base Unit**: 8px grid
- **Margins**: 24px minimum
- **Gutters**: 16px between elements
- **Columns**: 12-column responsive grid
- **Breakpoints**: 768px, 1024px, 1440px

**Hierarchy Rules**:
1. **Primary Message**: Largest, brand color
2. **Supporting Points**: Medium, neutral color
3. **Details**: Smallest, gray color
4. **CTAs**: Contrasting color, prominent placement

### Animation Guidelines

**Timing**:
- **Micro-interactions**: 200ms
- **Transitions**: 300ms
- **Complex animations**: 500ms
- **Loading states**: 1000ms+

**Easing**:
- **Standard**: ease-in-out
- **Entrances**: ease-out
- **Exits**: ease-in
- **Bounces**: cubic-bezier for playfulness

## Usage Guidelines

### Platform Adaptations

**Social Media**:
- **Instagram**: Square (1080x1080px)
- **Facebook**: Landscape (1200x630px)
- **Twitter**: Landscape (1024x512px)
- **LinkedIn**: Landscape (1200x627px)
- **Stories**: Vertical (1080x1920px)

**Email Marketing**:
- **Header**: 600px width maximum
- **Inline**: 300px width maximum
- **Footer**: 600px width maximum
- **Mobile**: Responsive scaling

**Web Usage**:
- **Hero Sections**: Full-width responsive
- **Content Blocks**: Container-width
- **Sidebars**: 300px fixed width
- **Modals**: Centered, responsive

### Quality Standards

**Resolution Requirements**:
- **Print**: 300 DPI minimum
- **Web**: 72 DPI standard, 144 DPI retina
- **Social**: Platform-specific optimization
- **Email**: 72 DPI, optimized file size

**File Formats**:
- **Web**: SVG preferred, PNG fallback
- **Print**: PDF or high-res PNG
- **Social**: JPG for photos, PNG for graphics
- **Email**: JPG optimized for size

### Brand Compliance

**Logo Usage**:
- **Minimum Size**: 100px width
- **Clear Space**: 2x logo height
- **Backgrounds**: Ensure sufficient contrast
- **Modifications**: No alterations allowed

**Voice & Tone**:
- **Professional**: Technical accuracy
- **Approachable**: Friendly explanations
- **Confident**: Clear value propositions
- **Trustworthy**: Honest representations

These infographic templates provide a comprehensive visual communication system that effectively conveys HeyZack's value propositions while maintaining brand consistency and professional presentation across all marketing channels and customer touchpoints.
