### Logo Usage Guidelines

<Tabs>
  <TabItem label="Primary Logo">
    
    #### HeyZack Primary Logo System
    
    **Logo Variations:**
    
    | Logo Type | Usage | Minimum Size | File Formats |
    |-----------|-------|--------------|-------------|
    | **Full Logo** | Primary brand mark | 120px width | SVG, PNG, PDF |
    | **Logo + Tagline** | Marketing materials | 150px width | SVG, PNG, PDF |
    | **Icon Only** | Social media, apps | 32px square | SVG, PNG, ICO |
    | **Wordmark** | Text-heavy layouts | 80px width | SVG, PNG, PDF |
    
    **Logo Construction:**
    - **Typography:** Custom wordmark based on Montserrat Bold
    - **Icon Element:** Stylized shield with modern geometric design
    - **Color Relationship:** Primary blue with accent orange
    - **Proportions:** 3:1 ratio (width to height) for full logo
    
    **Logo Spacing & Clear Space:**
    - **Minimum Clear Space:** 0.5x logo height on all sides
    - **Recommended Clear Space:** 1x logo height for optimal impact
    - **Text Proximity:** Minimum 0.25x logo height from other text
    - **Edge Distance:** Minimum 0.5x logo height from page edges
    
    **Logo Colors:**
    
    | Application | Primary Color | Secondary Color | Background |
    |-------------|---------------|-----------------|------------|
    | **Full Color** | HeyZack Blue (#1E40AF) | Security Orange (#F97316) | White/Light |
    | **Monochrome** | Black (#000000) | - | White |
    | **Reverse** | White (#FFFFFF) | - | Dark backgrounds |
    | **Grayscale** | Dark Gray (#374151) | - | Any |
    
  </TabItem>
  
  <TabItem label="Logo Don'ts">
    
    #### Logo Usage Restrictions
    
    **Prohibited Logo Modifications:**
    
    ‚ùå **Never Do These:**
    - Stretch, skew, or distort the logo proportions
    - Change the logo colors outside approved palette
    - Add effects like drop shadows, gradients, or outlines
    - Rotate the logo at any angle
    - Place logo on busy or low-contrast backgrounds
    - Use outdated or unofficial logo versions
    - Recreate or redraw the logo elements
    - Use the logo smaller than minimum size requirements
    
    **Background Restrictions:**
    - Avoid placing logo on images without sufficient contrast
    - Don't use logo on patterns or textured backgrounds
    - Ensure minimum 4.5:1 contrast ratio for accessibility
    - Test logo visibility at actual usage sizes
    
    **Color Restrictions:**
    - Never use colors outside the approved brand palette
    - Don't use low-opacity versions unless specified
    - Avoid color combinations that reduce readability
    - Test color combinations for accessibility compliance
    
    **Spacing Violations:**
    - Don't crowd the logo with other elements
    - Never place text or graphics within clear space
    - Avoid edge-to-edge placement without proper margins
    - Don't use logo as a pattern or watermark
    
  </TabItem>

    | **sm** | 8px | Small spacing | Button padding, small gaps |

    | **lg** | 24px | Large spacing | Component margins, layout gaps |
    | **xl** | 32px | Extra large | Section padding, major spacing |
    | **2xl** | 48px | Section spacing | Between major sections |
    | **3xl** | 64px | Page spacing | Hero sections, major divisions |
    | **4xl** | 96px | Layout spacing | Page margins, major layouts |
    
    **Component Spacing Guidelines:**
    
    | Component | Internal Padding | External Margin | Notes |
    |-----------|------------------|-----------------|-------|
    | **Buttons** | 12px 24px | 8px | Comfortable touch targets |
    | **Cards** | 24px | 16px | Adequate content breathing room |

    | **Navigation** | 16px 24px | 0px | Clear navigation hierarchy |
    | **Sections** | 48px 32px | 32px | Clear content separation |

    - Base line height: 24px (1.5x 16px base font)

    - Sufficient color contrast in image text
    - Avoid text-heavy images when possible
    - Provide alternative formats for complex diagrams
